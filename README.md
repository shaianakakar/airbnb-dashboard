# 📊 Airbnb Listings Excel Dashboard

## 📌 Project Overview
This project is an interactive Excel dashboard built to analyze Airbnb listing data. It provides insights into pricing trends, neighbourhood performance, room types, and host characteristics (Superhost vs Regular Host). The dashboard helps in understanding how different factors impact pricing and customer ratings.

---

## 🎯 Objectives
- Analyze pricing distribution across listings
- Compare performance of Superhosts vs Regular Hosts
- Identify top-performing neighbourhoods
- Understand room type distribution
- Explore review patterns and ratings

---

## 🛠 Tools & Features Used
- Microsoft Excel
- PivotTables
- Slicers
- Charts (Bar, Pie, Column)
- Data Cleaning & Transformation

---

## 📊 Dashboard Features
- Interactive slicers for dynamic filtering
- Price distribution analysis using buckets
- Neighbourhood-wise price comparison
- Room type breakdown
- Host type (Superhost vs Regular Host) comparison
- Review and rating insights

---

## 📁 Dataset
The dataset used includes Airbnb listing information such as:
- Price
- Neighbourhood
- Room type
- Host status
- Reviews and ratings

> Note: Due to file size limitations, the full dataset is not included.
